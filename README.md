# Moon Animator

Moon Animator is a Roblox Animating Simulation App, You can animate a R6 rig including R15.

Animating in this, not-well made studio can be **tricky**, but it takes time.

## Features

- Export JSON [Go to https://convert3d.org/ to import your animation into **Roblox Studio**]
- Plugins [Allow you is **customize** and **modify** things in the website, Like UI, or how you animate.]
- VFX [You can add VFX to your **animations** and you can choose when it should **activate** or **appear** and where it should go, in your animations]
- Smoother [Smooth your animations by pressing "M" while selecting the keyframes you want to smooth.]
